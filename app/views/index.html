This is the prototype for a new tool, an interim solution allowing users of Application Processing (AP) to find and remove applications that have not been started yet.

Version 1

Inital mockup based on conversations with AJ, Emily, and Warren.

Allows users to:

  • find an application by Application Barcode Reference (ABR)
  • view the status of the application
  • choose a location and team to assign to if the application has not been started

Find options in the prototype:

  • ABC123456 is an application that can be removed
  • 654321CBA is an application that cannot be removed
  • any other entry results in a no match error
  • blank shows an error message
