Prototype for a tool that allows users of Application Processing (AP) to find and remove applications that have not been started.

Version 4 - design complete

Design by Andy Porter, Mo Rowlands, Feb 2022

Updated to:

  • remove picking an office and e-serv queue
  • inform users more clearly whats happening, now the office and queue choices have gone
  • making the content clearer, in general (initially this tool was needed quickly and for an interim period, but will be around for at least another year)

example data to use

  • A123XYZ can be removed
  • B123XYZ cannot be removed
  • any other entry results in a no match error
  • blank shows an error message

All the pages in the tool:

  • find - find an application by Application Barcode Reference (ABR)

    error messages:

    • no ABR provided - use blank
    • no application with this ABR in AP - use a random value
  • can-remove - result page for we can remove from AP
  • cant-remove - result page for we cannot application remove from AP - as it is in progress
  • confirmation - confirmation its been sent to workflow
  • error - if there is a problem with the tool

Questions addressed

  • How quickly does the application appear in workflow?
    Answer: a few minutes
  • What queue does it go to in workflow... could we be clearer than just saying - the application will appear in workflow
    Answer: we've not been able to come up with a better form of words
  • What should caseworkers do if the application can't be removed - as it's in progress.
    Answer: create a stand-alone PAS note to stop an application being marked off
  • Are there other pages/content in the production version that don't appear in this prototype, that we need to address? We noticed some differences between screenshots sent by email, and this prototype.
    Answer: No - After asking, no one has said yes
  • What do we know about current user behaviour using the tool?
    Answer: there's been no research with people using the tool. We've learnt Workflow has a search ABR function so people can find the application immediately after removing it
